
# SOS - sbw's Operating System
It's just a toy for learning OS technology.

# Build from source

## Build dependencies
- g++
- cmake
- grub-mkrescue
- xorriso

## Build
```shell
mkdir build
cd build
cmake ..
make
```

## Runtime dependencies
- QEMU

## Run
```shell
make launch-qemu
```
