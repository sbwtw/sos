
# SOS - sbw's Operating System
It's just a exercise for learning OS technology.

# Build from source

## Build dependencies
- g++
- grub-mkrescue
- xorriso

## Build
```shell
make sos.iso
```

## Runtime dependencies
- QEMU

## Run
```shell
make run
```
